Typografie

font-family-primary

deze font wordt gebruikt als de standaard font op de pagina.

font-title

Deze font wordt alleen gebruikt voor de h1 elementen.

font-details-title

deze font wordt gebruikt voor de sub titels.

font-overzicht-title

op de overzicht pagina wordt dit als een titel gebruikt voor de informatie kaartjes

font-nav-headings

deze font wordt gebruikt in de navigatie menu als een titel. (hieronder komen linkjes)

font-nav-links

deze font wordt gebruikt voor linkjes in de navigatie menu
font-nav-headings deze font wordt op de overzicht pagina gebruikt voor de linkjes binnen de in formatie kaarjes.

Formulieren

button-nfo

Deze class wordt gebruikt voor de "nieuwe fouten ontdekt" buttons.

button-inaccessible

Deze class wordt gebruikt voor de "ontoegankelijk" buttons.

button-inaccessible

Deze class wordt gebruikt voor de "toegankelijk" buttons.

button-filter

deze class wordt gebruikt voor de filter buttons op de details pagina, die niet aangeklikt zijn.

button-filter-active

deze class wordt gebruikt voor de filter buttons op de details pagina, die active zijn.
